# Trivia Game

### Overview
This Trivia Game uses mainly JavaScript. Each question is timed and each answer is stored. At the end of the game, the score is displayed as well as a button that lets the player play the game again. 

### Technology Used
* html
* css
* bootstrap
* javascript
* jquery
